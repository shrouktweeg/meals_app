🚀 **Meal App – Flutter Project (Updated Version)**

I’m excited to share an updated version of my Meal App, originally built as part of my Flutter learning journey.

This project started with a focus on local storage, and I’ve now enhanced it by integrating APIs to make it more dynamic and closer to real-world applications.

---

##  Features

* **Onboarding Screens**

  * Built using Carousel Slider
  * Includes Smooth Page Indicator
  * Saves user state with Shared Preferences

* **Home Screen**

  * Displays meals dynamically
  * Uses FutureBuilder
  * Supports both local database (Sqflite) and API data

* **Add Meal Screen**

  * Add and store custom meals

* **Meal Details Screen**

  * View detailed information about each meal

* **Image Loading**

  * Optimized with Cached Network Image

---

##  What’s New

* Integrated REST API for fetching meals
* Combined local storage with remote data sources
* Implemented API handling using Dio
* Added Pretty Dio Logger for better debugging and network inspection
* Improved app scalability and data consistency
* Better handling of asynchronous operations
* More production-ready architecture
